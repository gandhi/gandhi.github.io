SmartSpenders
Finalish - Daemon Dash Hackathon 2015 (09/15)
Monitor family spending.
What started as a complete joke ended up asan actual chrome extension on the store. It uses Google Chrome's API and basic JavaScript to look for phrases that flag the page as a shopping page, and the program closes them. This was a finalist in the Daemon Dash Hackathon 2015.
